WPI CSA, iOS and Android App written for Chinese Student Association of WPI

Supports a forum that can be used to host events, trade and post feeds. Integrated Braintree for ticket charging service. Presents the book Survival Guide for Chinese Students using html and SQLite.The server side uses Java Restful and Spring MVC. This app is published on both App Store and Google Play Store.
